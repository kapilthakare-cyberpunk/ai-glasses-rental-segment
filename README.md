# AI Glasses Rental Segment Research

Generates branded HTML research reports for AI/smart glasses rental businesses.

## What It Does

- Analyzes AI glasses models (Ray-Ban Meta, etc.) for rental viability
- Generates professional HTML reports with pricing, specs, and rental projections
- Outputs ready-to-share research documents

## Reports Include

| Section | Content |
|---------|---------|
| Model Comparison | Camera, battery, weight, connectivity specs |
| Pricing Analysis | Retail price + rental rate projections (conservative/premium) |
| Market Position | Category fit, pros/cons, availability |
| Rental Viability | Daily rate estimates, ROI projections |

## Usage

```bash
python3 generate_reports.py
```

Reports are saved to `reports/` directory.

## Data Source

- `Pricelist_20260708.csv` — Current rental pricing data
- `images/` — Product images for reports

## Tech Stack

- Python 3.x
- HTML templates
- CSV data processing
